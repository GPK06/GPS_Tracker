# SERVICE EMPLOYEE TRACKER
To design and build a web based application that allow employers to track their users location and store the coordinates in a database.
This helps in having their information on what they do and where they are at to maintain a professional watch on their employees.
The app utilizes Flask as the framework

Features:-
*   *Realtime Tracking (Coordinates)*
*   *Immediate OFFLINE Status Detection*
*   *Visual Dashboard (For admins)*
*   *Permanent Data Storage (local file)*

Structure:-
*   **Backend:** Python, Flask
*   **Frontend:** HTML, JavaScript
*   **Database:** SQLite
