# GPS TRACKER
To design and build a web based application that allow users to track others' location and store the coordinates in a database.
The app utilizes Flask as the framework

Features:-
*   *Realtime Tracking (Coordinates)*
*   *Immediate OFFLINE Status Detection*
*   *Visual Dashboard (For admins)*
*   *Permanent Data Storage (local file)*

Structure:-
*   **Backend:** Python, Flask
*   **Frontend:** HTML, JavaScript
*   **Database:** SQLite
